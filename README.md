# AI SaaS Landing Page (Claude Inspired)

A modern AI product landing page inspired by Claude. This project was built to practice advanced frontend UI techniques used in modern SaaS websites.

## Features
- Responsive design (mobile + desktop)
- Custom animated cursor
- Scroll reveal animations using Intersection Observer
- Animated chat UI simulation
- Modern UI using CSS variables
- Smooth hover and transition effects

## Tech Stack
HTML  
CSS  
JavaScript

## Purpose
To explore modern landing page design patterns used in AI products and SaaS platforms.
